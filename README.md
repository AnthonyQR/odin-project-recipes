# odin-project-recipes
Practicing HTML and CSS through The Odin Project by creating a recipe website

With HTML, I learned how to:
- Setup HTML documents, including the \<html\>, \<head\>, and \<body\> tags
- Add text using \<h#\> and \<p\> tags
- Add links to other pages using \<a href\> tags
- Add images using \<img\> tags
- Create lists using \<ul\>, \<ol\>, and \<li\> tags

With CSS, I learned how to:
- Create a navbar using display: flex & HTML lists
- Utilize ids and classes to apply styles to certain elements
- Utilize divs to apply styles to groups of content
- Utilize margins, text-align & justify-content to position elements

Link to website: https://anthonyqr.github.io/odin-project-recipes/

Link to The Odin Project: https://www.theodinproject.com/lessons/foundations-recipes
